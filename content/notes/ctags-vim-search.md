Vim uses file called `tags` in the current directory.

* `vim -t tag`

Start vim and position cursor at file and line where "tag" is defined.

* `:ta tag`

Find tag

* `Ctrl-]`

Find tag under cursor

* `Ctrl-T`

Return to previous location before jump to tag
