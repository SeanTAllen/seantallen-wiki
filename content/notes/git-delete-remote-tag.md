title: "Remove a tag in a remote Git repository"
slug: "git-delete-remote-tag"

tags: [Git]
description: "How to delete a tag in a remote repository."
---
```bash
git push --delete REMOTE_NAME TAGNAME
```

for example:

```bash
git push --delete origin mytag
```
